ブート遅い件、FPGA(XC7A35T)のコンフィグ完了を待ってからMCU起動してませんか？ DONE pin の立ち上がりは平均 220ms くらいなので、並列に動かせば 90ms くらい削れるはず。 
// boot_seq.c (現在: 直列)
fpga_load_bitstream();   // ~220ms (blocking)
mcu_peripherals_init();  // ~90ms

// 提案: DMA で並列ロード
fpga_load_bitstream_async();    // 別スレッドで開始
mcu_peripherals_init();         // この間に並行
fpga_wait_done();               // join
