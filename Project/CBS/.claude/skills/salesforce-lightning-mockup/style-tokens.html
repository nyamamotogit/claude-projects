グローバル UI (公式)

slds-global-header_container             … ヘッダー外殻
slds-global-header                       … ヘッダー本体 (slds-grid で内部レイアウト)
slds-global-header__item                 … ヘッダー内アイテム (logo / search / actions)
slds-global-header__item_search          … 検索バー専用 (中央寄せ)
slds-global-actions                      … 右側アクション群 (ul)
slds-global-actions__item                … アクション 1 個 (li)
slds-global-actions__favorites           … お気に入りボタン (青背景・白アイコン自動)
slds-notification-badge                  … 通知の赤丸バッジ (数字付き)
slds-context-bar                         … Global Navigation 親
slds-context-bar__primary                … 左側 (App Launcher + App Name)
slds-context-bar__item_app               … App Launcher の修飾子
slds-context-bar__app-name               … App 名 (太字・大きめ)
slds-context-bar__secondary              … 右側ナビリンク群 (nav)
slds-context-bar__item                   … nav item 1 個 (li)
slds-context-bar__label-action           … クリッカブルなテキストリンク
slds-context-bar__icon-action            … アイコンボタン (chevron dropdown 等)
slds-is-active                           … アクティブな item (青下線 #0176d3 3px が自動)

レイアウト

slds-grid slds-wrap                      … flex 並び (折返しあり)
slds-grid slds-gutters_small             … gap 0.5rem (推奨)
slds-grid slds-gutters                   … gap 1.5rem
slds-grid_align-spread                   … justify-content: space-between
slds-grid_vertical-align-center          … align-items: center
slds-col slds-size_2-of-3                … 幅 2/3
slds-col slds-size_1-of-3                … 幅 1/3
slds-col slds-size_1-of-4                … 幅 1/4 (4 カラム KPI 用)
slds-p-around_medium                     … padding 全方向 (12px)
slds-m-bottom_large                      … margin-bottom (24px)
slds-card                                … 白い角丸カード
slds-card__header / slds-card__body      … カード内構造

プログレスバー (横棒グラフ用)

slds-progress-bar                        … 親 (ベース)
slds-progress-bar_large                  … 高さ 1rem (デフォルト 0.5rem)
slds-progress-bar__value                 … 内部の塗り (青)
slds-progress-bar__value_success         … 受注/成功カラー (緑)

実装例は templates/ と components/ を参照。 SLDS にない要素 (Service Console の 3 ペイン、Kanban カラム外枠、Agentforce パネル、Slack 埋め込み) のみ 自前 sf-* クラスで補完する。詳細は SKILL.md の「自前 sf-* を使ってよい場所 ホワイトリスト」を参照。
